# SONAR Rock vs Mine Prediction

A Machine Learning project that predicts whether a sonar signal is bouncing off a Rock or a Mine.

## Algorithm Used
- Logistic Regression (Binary Classification)

## Libraries
- NumPy
- Pandas
- Scikit-learn

## Dataset
- 208 samples, 60 sonar frequency features
- Labels: R (Rock) or M (Mine)

## How to Run
1. Clone the repo
2. Install dependencies: `pip install numpy pandas scikit-learn`
3. Open the `.ipynb` file in VS Code or Jupyter
4. Run all cells
